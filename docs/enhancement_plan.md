# 功能完善与拓展规划 (v3.2)

根据项目现状及“数据展示、分析深度”的需求，制定以下功能增强计划。

## 1. 数据展示升级 (Data Visualization)

### 1.1 仪表盘 (Dashboard)
- **市场情绪仪表盘**: 引入 ECharts 仪表盘，展示当前市场恐慌/贪婪指数（基于涨跌家数比或 AI 综合评分）。
- **行业热力图**: 展示当前领涨/领跌的行业板块（需后端聚合数据，初期可模拟）。
- **快捷入口优化**: 增加“最近访问”的股票列表。

### 1.2 个股详情 (Stock Detail)
- **基本面数据栏**: 在 K 线图上方增加关键指标跑马灯或卡片：
  - 市盈率 (PE-TTM) / 市净率 (PB)
  - 总市值 / 流通市值
  - 换手率 / 量比
  - 52周最高/最低
- **技术指标增强**:
  - 为 K 线图增加 MACD、RSI、BOLL 等常见技术指标子图（Overlay 或 Subchart）。
  - 支持 K 线周期切换（日/周/月/分钟）。

## 2. 分析深度拓展 (Deep Analysis)

### 2.1 AI 深度诊断 (AI Deep Diagnosis)
- **结构化输出**: 升级 `StockDetail` 的 AI 分析 Prompt，使其返回 JSON 格式数据。
- **可视化报告**: 将 AI 纯文本分析转化为可视化报告：
  - **雷达图**: 估值、成长、盈利、资金、技术 5 维评分。
  - **关键点位**: 自动提取支撑位、压力位，并在 K 线图上标注（进阶）。
  - **操作建议**: 明确的 买入/卖出/观望 信号及置信度。

### 2.2 策略回测可视化 (Backtest Viz)
- **收益曲线对比**: 将策略收益率与基准指数（如沪深300）叠加对比。
- **月度回报热力图**: 展示每个月的盈亏情况。

### 2.3 Vision 模块增强
- **历史记录**: 保存用户的图片分析记录，方便回溯。
- **一键自选**: 分析结果中识别出的股票，支持一键加自选。

## 3. 交互与体验 (UX)
- **移动端适配**: 进一步优化 K 线图在手机端的触摸交互（缩放、平移）。
- **骨架屏**: 数据加载时使用 Skeleton Screen 替代简单的 Loading Spinner。

---

**建议优先执行路径**:
1.  **StockDetail 基本面数据展示** (提升信息密度)
2.  **StockDetail AI 诊断结构化** (提升分析深度，复用 Vision 模块的经验)
3.  **Dashboard 市场情绪图表** (提升首页视觉冲击力)
